# Research Notebooks

These notebooks and figures are report-support artifacts. They are not required
by the runtime pipeline, tests, Streamlit demo, or held-out evaluation.

The notebooks were originally developed during exploratory analysis. Some cells
assume execution from the repository root and some generated intermediates, such
as `data/processed/items.csv`, are intentionally ignored because they are large
regenerable artifacts. Treat the notebooks as analysis provenance unless a
specific figure/table needs to be regenerated.

For final writing, prefer the consolidated Markdown and figure outputs:

- `eda_report.md`
- `figures/`
- `docs/evaluation_design.md`
- `docs/cluster_interpretation.md`
- `docs/ablation_study.md`
